= StorageGRID S3 REST API가 충돌하는 클라이언트 요청을 해결하는 방법
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
동일한 키에 쓰는 두 클라이언트 등의 충돌하는 클라이언트 요청은 "최신 성공" 기준으로 해결됩니다.

"Latest-WINS" 평가 시기는 S3 클라이언트가 작업을 시작할 때가 아니라 StorageGRID 시스템이 지정된 요청을 완료하는 시점을 기준으로 합니다.
