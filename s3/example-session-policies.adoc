= StorageGRID S3 REST API 세션 정책 예
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
다음 예를 사용하여 StorageGRID 세션 정책을 구축하세요.



== 예: 객체 검색을 허용하는 세션 정책 설정

이 예시에서 세션의 주체는 bucket1에서만 객체를 검색할 수 있습니다. link:../s3/use-access-policies.html#use-putoverwriteobject-permission["S3:PutOverwriteObject"] 권한 사용과 같은 StorageGRID 관련 작업을 제외한 모든 다른 작업은 암묵적으로 거부됩니다. 세션 정책은 AssumeRole API를 호출할 때 JSON 파일로 제공할 수 있습니다.

[listing]
----
{
  "Statement": [
    {
      "Action": "s3:GetObject",
      "Effect": "Allow",
      "Resource": "arn:aws:s3:::bucket1/*"
    }
  ]
}
----
